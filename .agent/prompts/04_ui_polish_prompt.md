# 04_ui_polish_prompt.md - UI Polish Protocol

Use this prompt when improving visuals giving "Soft Liquid Glass" feel.

## 1) Scope
- **Target**: Specific component or page.
- **Design System**: Strict adherence to `packages/ui` tokens.
- **No New Libs**: Do not install new CSS libraries.

## 2) Plan
- **Visual Goal**: What should it look like? (Glassmorphism, padding, motion).
- **Accessibility**: Ensure contrast and focus states.

## 3) Task List
- [ ] Apply tokens
- [ ] Add motion (if needed)
- [ ] Check dark mode
- [ ] Check mobile responsiveness

## 4) Evidence
- **Screenshots**: Mandatory Before/After.
