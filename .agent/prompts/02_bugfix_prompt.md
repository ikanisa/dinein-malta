# 02_bugfix_prompt.md - Bugfix Protocol

Use this prompt when fixing a reported bug.

## 1) Analysis
- **Reproduction**: Steps to reproduce the bug.
- **Root Cause**: Why is it happening?
- **Scope**: Isolate the fix. Do not refactor unrelated code.

## 2) Plan
- **The Fix**: Minimal viable change.
- **Regression Test**: How to ensure it doesn't break again?
- **Verification**: How to verify the fix works?

## 3) Task List
- [ ] Reproduce issue
- [ ] Apply fix
- [ ] Verify fix
- [ ] Check for regressions

## 4) Evidence
- **Before/After**: Screenshots or logs.
